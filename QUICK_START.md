# Quick Start Guide - Demo Ready

## За 5 минут до демо

### Шаг 1: Установить Extension (2 минуты)

1. Откройте Chrome → `chrome://extensions/`
2. Включите "Developer mode" (правый верхний угол)
3. Нажмите "Load unpacked"
4. Выберите папку: `chrome-extension/`
5. Расширение установлено!

### Шаг 2: Настроить OpenAI Key (1 минута)

1. Кликните на иконку расширения в Chrome
2. Вкладка "Settings"
3. Вставьте ваш OpenAI API Key (формат: `sk-...`)
4. Нажмите "Save Settings"

### Шаг 3: Проверить работу (2 минуты)

1. Откройте любую страницу Jira (или просто Extension popup)
2. Extension → вкладка "Generate"
3. Введите: "Create login page with form validation"
4. Нажмите "Generate Task Preview"
5. Должен появиться предпросмотр задачи

**Готово! Система работает.**

## Для полного функционала (опционально)

### Запустить Backend (если нужен анализ проектов)

```bash
# В терминале:
cd "jira plugin"
npm install
cp env.example .env

# Отредактировать .env:
# JIRA_BASE_URL=https://your-jira.atlassian.net
# JIRA_USERNAME=your-email@example.com
# JIRA_API_TOKEN=your-token
# JIRA_PROJECT_KEY=PROJ
# OPENAI_API_KEY=sk-...

npm run dev
```

Backend запустится на `http://localhost:3000`

## Troubleshooting

**Extension не работает:**
- Проверьте OpenAI ключ в Settings
- Перезагрузите расширение (chrome://extensions/)

**Не заполняется форма в Jira:**
- Убедитесь что открыта страница создания задачи
- Обновите страницу Jira

**Backend не запускается:**
- Проверьте .env файл
- Убедитесь что все переменные заполнены
- Проверьте порт 3000 (не занят ли)
